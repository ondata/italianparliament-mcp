Gli strumenti

Parlamentari, disegni di legge, votazioni, emendamenti, interventi, commissioni, interrogazioni: ogni strumento è insieme un comando CLI e un tool MCP. L'elenco qui sotto è letto in tempo reale dal server.

Tre modi di usarlo

Stessi dati, tre porte d'ingresso. La CLI per pipeline ed export, il server MCP per parlare coi dati in linguaggio naturale, le skill per insegnare a un agente il mestiere.

01 · Da terminale

CSV e JSONL pronti per l'analisi. Filtri per legislatura, date, parole chiave.

npm install -g @aborruso/italianparliament-mcp
italianparliament guide

      

    


    

      
02 · Da Claude (MCP)

      
Server remoto già in piedi su Cloudflare: nessuna installazione, si aggiunge l'endpoint e si comincia a fare domande.

      
{
  "mcpServers": {
    "italianparliament": {
      "url": "https://italianparliament-mcp
            .andy-pr.workers.dev/mcp"
    }
  }
}

      
— lo stesso endpoint che sta alimentando questa pagina

    


    

      
03 · Con le skill

      
Due Agent Skill incluse nel repo insegnano all'agente il flusso di lavoro: scoperta → URI → dettaglio, catene tra comandi.

      
npx skills add aborruso/italianparliament-mcp \
  --skill italian-parliament-cli
npx skills add aborruso/italianparliament-mcp \
  --skill italian-parliament-mcp
